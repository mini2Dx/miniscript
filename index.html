Easily add scripting to your Java games

With miniscript you can easily add scripting to your games in any of the following languages:

  • Groovy
  • Lua
  • Python
  • Ruby

What would you use scripting for?

  • Scripting cutscenes
  • Adding a mod/scripting API
  • Implementing dynamic AI systems
  • and more!

Sounds good? Check out the wiki to get started.
